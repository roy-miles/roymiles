Problem formulation

We have a set of features represented as a p dimensional vector. We wish to find a linear mapping from these features to a real number output. This problem can then be used, for example, to estimate the price of housing based on the location, size, number of rooms etc of the house (known as the features). In the case of one feature, the relationship between the input and output variable will simply be $ y = mx + c $. This can be further simplified by using vector notation and encorporated the y-intercept into $ x $ $ y = [1 x][c ; m]$ By generalising to p features and n outputs, the result is a matrix equation, where X is a ... $ Y = BX $
